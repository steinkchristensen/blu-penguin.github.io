Wolf Robotics Project

Welcome! My name is Kaden Strand.

Abstract

This project will provide an efficient and repeatable method to accurately calibrate a robotic arm in three dimensional space using a laser sensor. Once we have completed our primary goal of calibration, we will then explore the possibilities of using the sensor in a variety of welding applications. These include creating search routines to find specific points that allow the robot to accurately weld based on the sensor data.

Project Background

Wolf Robotics is a systems integrator for industrial welding applications using 6-axis robots. The project aims to improve the process of detecting weld features by attaching a laser spot sensor to a robotic arm. By combining robotic motion with distance data from the laser sensor, the team will be creating a system that is capable of detecting features in three-dimensional space. To accomplish this, the robot and the sensor must be calibrated together.

This project addresses a specific industry problem at Wolf Robotics. In this project we are creating a calibration process that will allow the use of a laser spot sensor as a feature detection tool. Facilitating the use of a laser spot sensor in the welding process will decrease the total weld time and increase the total welding output from Wolfâ€™s robotic systems. The laser sensor is cheaper than existing optic sensing tools, yet quicker and more accurate than conventional tactile sensing techniques.

Project Goals and Constraints

Objectives
  • Devise a method for determining a TCP for a laser spot sensor mounted on a robot.
  • Create a program to implement this method for Wolf robotic systems.
  • Use the calibrated sensor to search for points of interest on specific weld joints.
Final Design

The final design will include a program written in RAPID, an ABB programing language, which will automatically calibrate and find the user-defined Tool Center Point of the Keyence IL-300 Laser Sensor. It will also include a physical feature that the program will use to calibrate the system. An extension to the project may include a RAPID program that uses the laser sensor to find features and points of interest.

Design Constraints
  • Use an ABB 6 axis robot and a Keyence IL-300 Laser Sensor.
  • Programs must be written in ABB programming language, RAPID.
  • Calibration method must be accurate, efficient and repeatable.
  • Calibration method must be useable on all ABB robots, including different sized robots.
